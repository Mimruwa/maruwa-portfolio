# Maruwa Portfolio Website

This is a static portfolio website built with HTML, CSS, and JavaScript.

## Files
- `index.html` — main website content
- `assets/styles.css` — website design and responsive layout
- `assets/script.js` — mobile menu and footer year

## How to use
1. Open `index.html` in any browser to preview the website.
2. Edit the text in `index.html` to update sections.
3. Upload the whole folder to GitHub Pages, Netlify, Vercel, or any hosting provider.

## Suggested updates before publishing
- Add ORCID, ResearchGate, Google Scholar, and Web of Science links if available.
- Add a downloadable CV PDF link if desired.
- Confirm the exact expected graduation year.
